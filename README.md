# CarBlog


Staticky web, autovy blog: HTML/CSS/JS, galeria importovana.  (GLightbox), garazova tabulka nacitana z js/cars.json cez fetch.
Dizajn: rucnyDesignHarmady.pdf v roote.


## Spustenie lokalne

Potreba otvorit projekt cez nejaky localhost, file:// nebude fungovat. Neotvarat subory z prehliadaca. VsCode extension localhost, webstorm vpravo hore alebo python3 -m http.server 8080.
potom otvorit:
http://localhost:8080/html/index.html
